5.1 Basics of bar charts

These uses the geom_col() function to display values in the data, geom_bar() to count the cases in a group.

In this example, average highway miles per gallon (hwy) is displayed by a the type of car (class).

(hwy_class <- mpg %>% 
  group_by(class) %>% 
  summarise_at(vars(hwy), list(average_hwy= ~round(mean(.x), digits = 1))))

5.1.1 A basic bar chart

That data is then fed to ggplot to produce a simple bar chart.

ggplot(data=hwy_class) + 
  geom_col(mapping=aes(x=class, y=average_hwy))

5.3 Adding labels to data

The geom_text function is what adds the labels:

  • had to move the aes() bit out of the geom_bar() bit to make this work.
  • the vjust bit moves it above and below the bar. A lower minus number would be higher above the bar.
ggplot(data=hwy_class) +
  aes(x=class, y=average_hwy) +
  geom_bar(stat="identity") + 
  geom_text(aes(label=average_hwy), vjust=-1)


Here the label is below the bar, vjust=1.5. The colour has been switched to white and the size increased also.

ggplot(data=hwy_class) +
  aes(x=class, y=average_hwy) +
  geom_bar(stat="identity") + 
  geom_text(aes(label=average_hwy), vjust=1.5, colour="white",  size=6)

5.3.0.1 Adding them to our dodged plot above.

We also have to ‘dodge’ the position of the labels within geom_text, again in the position argument.

plot + 
  geom_text(aes(label=average_hwy_year), vjust=1.6, color="white",
            position = position_dodge2(0.9), size=3.5)



5.4 Other formatting


5.4.1 Bar width

Amend bar width within geom_col argument with ‘width’.

The default is 0.9, smaller values make bars smaller. Max = 1 and removes all space between bars.

If you make the entire graph wider or narrower, the bar dimensions will scale proportionally.

more_space <- ggplot(data=hwy_class) +
  geom_col(mapping=aes(x=class, y=average_hwy), width=0.9)

less_space <- ggplot(data=hwy_class) +
  geom_col(mapping=aes(x=class, y=average_hwy), width=0.5)

ggarrange(more_space, less_space, ncol = 1)


5.4.2 Change colours

In the below the colour affects only the outline, without the fill the bars would be dark grey still.

ggplot(data=hwy_class) +
  geom_col(mapping=aes(x=class, y=average_hwy), colour="blue", fill="white", width = 0.5)


Or bar colour can be changed by groups.

  • Add colour=var, where var is the variable you want to vary the colours by.
ggplot(data=hwy_class) +
  aes(x=class, y=average_hwy, colour=class) +
  geom_col(fill="white", width=0.5)


5.4.3 Apply themes

p <- ggplot(data=hwy_class) +
  geom_col(mapping=aes(x=class, y=average_hwy), fill="steelblue") 

p + theme_minimal()

5.6.0.1 Barplot of counts

Similar to a histogram, the stat=“bin” command allows ggplot() to work like table(), but specifically for numeric values.

ggplot(data=mpg) +
  aes(x=cyl) +
  geom_bar(stat="bin")
`stat_bin()` using `bins = 30`. Pick better value `binwidth`.


Or with character values, try geom_bar without an argument.

ggplot(data=mpg) +
  aes(x=class) +
  geom_bar()

dynamic_str_wrap <- function(text, plot_width = 7, num_labels = 1, char_width = 0.1, margin = 0.1) {
  max_label_width <- (plot_width * (1 - margin)) / num_labels
  max_chars <- max_label_width / char_width
  str_wrap(text, width = floor(max_chars))
}

wrap_titles <- function(title, plot_width = 7, char_width = 0.09) {
  max_chars <- floor(plot_width / char_width)
  wrapped_title <- str_wrap(title, width = max_chars)
  return(wrapped_title)
}

freq <- function(dat, var) { 

x_var <- as.character(substitute(var))

table <- dat %>%
  filter(!is.na({{var}})) %>%
  group_by({{var}}) %>%
  summarise(n = n()) %>%
  mutate(p = 100 * (n / sum(n)))

table$x_label <- var_label(dat[[x_var]])

return(table)
  
}

mean_func <- function(dat, x, y) { 
  
  x_var <- as.character(substitute(x))
  y_var <- as.character(substitute(y))

  table <- dat %>% 
  filter(!is.na({{x}})) %>% 
  mutate(!!sym(x_var) := fct_drop(as_factor({{x}}))) %>%
  group_by({{x}}) %>%
  summarise(!!sym(y_var) := mean({{y}}, na.rm = TRUE), n = n())

  table$x_label <- var_label(dat[[x_var]])
  table$y_label <- var_label(dat[[y_var]])

  return(table)
  
}
